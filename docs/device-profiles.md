# Device Profiles

## Pocket

当前把 `Pocket` 视为第一阶段的核心拍摄来源之一。

关注点：

- 天空和高光层次
- 导出后是否丢失 HDR 感
- 与样片观感的距离

建议验证场景：

- 正午天空
- 室外高反差
- 逆光主体

## iPhone

当前把 `iPhone Pro` 视为另一条核心拍摄来源。

关注点：

- 高亮区域是否过早发白
- 肤色和亮度是否在发布后失真
- 手机与 MacBook 观感差异

建议验证场景：

- 人像肤色
- 室内窗边反差
- 夜景高光
